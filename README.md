# NKUChatBot WEB

## 技术说明

- 聊天机器人实现技术：通过模块化的处理方式，每个用户输入问题按顺序进入每个模块。

  每个模块定义一个是否可以成功回答的函数。如果可以成功回答，则成功回答，否则进入下一个模块。
  
- 后端使用 Django 的 Python 框架，予以数据库存储支持和聊天机器人模块。

- 前端使用纯 JavaScript 语言，通过 OOP 的方式实现字母池的动态效果（详细见后文 bilibili 演示视频）
